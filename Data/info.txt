Common Technical Troubleshooting (Internal Context)Q: What happens if the API rate limit is exceeded?A: Our standard API gateway (Kong or AWS API Gateway) limits requests to 1,000 per minute per IP. If exceeded, the user receives a 429 Too Many Requests error. This limit can be lifted for enterprise clients via the X-API-KEY header configuration.Q: How do you handle image optimization?A: All images uploaded to our CMS are automatically processed via Cloudinary or AWS Lambda to generate WebP formats. If a client sees "blurry" images, check if they are uploading below our minimum resolution threshold of 1920x1080 for hero banners.Q: Why is the deployment failing on the staging server?A: Common reasons documented in our runbooks:Environment Variables: Check if .env file is missing keys for STRIPE_SECRET_KEY or DB_HOST.Linting Errors: The pipeline will auto-reject commits that fail ESLint or Prettier checks.Migration Conflicts: Check if a Django migration file was deleted manually without rolling back the DB.8.
